# Overpay

This practical exercise involves transitioning from Figma to a React app.


## Run Locally

Clone the project

```bash
  git clone https://github.com/AadarshGhodasara/Overpay.git
```

Go to the project directory

```bash
  cd Overpay
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
